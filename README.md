
# 🚀 MTProxy 一键安装脚本备份

本项目是对 MTProxy 一键安装脚本的备份。原脚本作者为 **rts600**。

## ✨ 脚本特性
- 🛡️ 抗封锁
- 🕵️‍♂️ 不会被检测
- 🔒 不会被封端口

> ⚠️ **注意**：由于原作者曾经删库，为防止再次出现此问题，特拉取本项目作为备份。亲测非常好用！👍

## 📥 安装命令

### 1. 使用本仓库备份脚本（推荐 ✅）
```
bash <(curl -sSL https://raw.githubusercontent.com/yanjie233/MTProxy/main/mtproxy.sh)
```

### 2. 使用原作者脚本
```
bash <(curl -sSL https://raw.githubusercontent.com/rts600/MTProxy/main/mtproxy.sh)
```
